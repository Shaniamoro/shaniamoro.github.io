Keyed Caesar Cipher

This module includes an implementation of the Keyed Caesar cipher. The Keyed Caesar Cipher involves encrypting messages through the use of an encryption "key". The key is just a word which is used to encrypt your message. More information on the Keyed Caesar Cipher can be found here.

Give It A Try!

The program accepts a users key and message and either encodes or decodes the message
